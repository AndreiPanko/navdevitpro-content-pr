# ISFILE Function (Variant)
Indicates whether a C\/AL variant contains a File variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISFILE  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value\/Return Value  
 Type: Boolean  
  
 **true** if the C\/AL variant contains a File variable; otherwise, **false**.
