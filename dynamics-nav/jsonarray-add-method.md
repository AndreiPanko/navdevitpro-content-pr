# Add Method
Adds a new value at the end of the **JsonArray**.

```
JsonArray.Add(Value)
```

## Parameters
*JsonArray*  
Type: JsonArray

*Value*  
Type: JsonToken | JsonObject | JsonValue | JsonArray | Boolean | Char | Byte | Integer | BigInteger | Decimal | Duration | String | Date | Time | DateTime
