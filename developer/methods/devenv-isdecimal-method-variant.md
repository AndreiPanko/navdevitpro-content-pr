# ISDECIMAL Method (Variant)
Indicates whether an AL variant contains a Decimal variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISDECIMAL  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **true** if the AL variant contains a Decimal variable; otherwise, **false**.
