# Codeunit Object

## Codeunit Example
This codeunit example  

```
codeunit 50040 CreateCustomer
{
    trigger OnRun();
    var
        r : record Customer;
    begin
        if not r.HasShoeSize() then
            r.ShoeSize :=  42;  
    end;
}

```
